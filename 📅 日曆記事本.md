# 📅 日曆記事本

一個簡潔優雅的網頁版日曆記事應用，記錄每一天的美好時刻。

## ✨ 功能特性

- **月曆介面** - 直觀的月份檢視，快速瀏覽整月記事
- **月份切換** - 輕鬆切換上下月份
- **文字記事** - 為每一天新增、編輯、刪除記事
- **標籤分類** - 用 5 種標籤對記事進行分類（工作、生活、學習、健康、旅遊）
- **全文搜尋** - 快速搜尋所有記事內容
- **深色模式** - 支援淺色/深色主題切換
- **本地儲存** - 所有資料儲存在瀏覽器，無需帳號登入
- **離線使用** - 無網路也能使用
- **響應式設計** - 完美適配手機、平板、電腦

## 🚀 快速開始

### 方式 1：GitHub Pages（推薦）

1. Fork 本倉庫
2. 在倉庫設定中啟用 GitHub Pages
3. 選擇 `main` 分支作為來源
4. 訪問 `https://你的用戶名.github.io/calendar-diary-github`

### 方式 2：本地使用

1. Clone 倉庫：
   ```bash
   git clone https://github.com/你的用戶名/calendar-diary-github.git
   ```

2. 用瀏覽器打開 `index.html`

### 方式 3：直接使用

訪問 GitHub Pages 部署的線上版本（如果已啟用）

## 📝 使用方法

1. **新增記事**：點擊日期或右下角的 `+` 按鈕
2. **編輯記事**：點擊已有記事的日期
3. **添加標籤**：在記事對話框中選擇標籤
4. **搜尋記事**：使用頂部搜尋欄搜尋關鍵字
5. **切換主題**：點擊右上角的主題按鈕

## 💾 資料儲存

所有記事資料都儲存在瀏覽器的 `localStorage` 中：
- 資料完全本地化，不上傳到任何伺服器
- 清除瀏覽器快取會刪除所有資料
- 建議定期備份重要記事

- 其他現代瀏覽器

## 🔧 自訂修改

編輯 `index.html` 中的以下部分：

- **標籤**：修改 `TAGS_LIST` 陣列
- **顏色**：修改 CSS 中的 `:root` 變數
- **儲存鍵**：修改 `STORAGE` 常數
