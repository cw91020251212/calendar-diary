# 📅 日曆記事本 Pro

一個簡潔優雅且功能強大的網頁版日曆記事應用，支援圖文混排與本地安全儲存，完美記錄每一天的美好時刻。

## ✨ 全新升級功能特性

- **滿版無縫日曆** - 直觀的月份檢視，自適應填滿螢幕，快速瀏覽整月記事。
- **圖文混排記錄** - 不僅能輸入文字，還能上載圖片。內建圖片自動壓縮引擎與「劇院級全螢幕放大鏡」。
- **12 色自訂標籤** - 提供橫向滑動的 12 色調色盤，用家可為每種顏色自訂專屬名稱（如：工作、私人、緊急），一目了然。
- **先閱後改體驗** - 點擊記事先進入「全螢幕閱讀模式」防誤觸，需要時才切換至「沉浸式全螢幕編輯器」。
- **資料安全中心** - 一鍵打包下載所有日記與圖片（.json 格式），支援電郵備份與跨裝置還原。
- **深色模式** - 支援淺色/深色主題無縫切換。
- **極致私隱保護** - 所有資料（包含圖片）100% 儲存在瀏覽器本地，無需連網，不上傳任何伺服器。

## 🚀 快速開始

### 方式 1：GitHub Pages（推薦）

1. Fork 本倉庫
2. 在倉庫設定中啟用 GitHub Pages
3. 選擇 `main` 分支作為來源
4. 訪問 `https://你的用戶名.github.io/你的專案名稱`

### 方式 2：本地使用 / 手機 App 化

1. 將專案下載到本地，直接用瀏覽器打開 `index.html`。
2. **手機用家強烈建議**：在 Safari 或 Chrome 打開網頁後，點擊「加入主畫面 (Add to Home Screen)」，即可將其當作原生 App 般全螢幕使用！

## 📝 使用指南

1. **新增記事**：點擊右下角的 `+` 懸浮按鈕。
2. **自訂標籤**：在編輯器上方滑動選擇顏色，並在下方輸入框自訂該顏色的代表意義。
3. **查閱與放大**：點擊日曆上的日期進入清單，再點擊卡片進入「閱讀模式」。點擊任何圖片即可全螢幕放大。
4. **備份資料**：點擊右上角的「⚙️ 齒輪」圖示，選擇「下載備份檔案」或「備份至私人電郵」。

## 💾 資料儲存與備份警告

- 核心資料庫已全面升級為 **IndexedDB**，以支援大容量圖片儲存。
- 資料完全本地化，絕對安全，但**清除瀏覽器快取會刪除所有資料**。
- 換手機或重設瀏覽器前，請**務必**進入右上角的「⚙️ 設定與備份」中心，下載您的 `.json` 備份檔。
